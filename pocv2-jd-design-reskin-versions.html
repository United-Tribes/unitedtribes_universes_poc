Critical Design Rules
No Gray Text
Every instance of #9ca3af, #6b7280, #4b5563 must go. Use #1a2744 (navy) for primary, #3d3028 (warm brown) for secondary.
No Pastels
Universe tiles and UI elements must use rich, saturated colors. No muted or washed-out tones anywhere.
Warm Everything
Every surface, border, shadow, and hover state must feel warm. Cool grays (#e5e7eb, #f3f4f6) are banned. Gold replaces blue for active states.
